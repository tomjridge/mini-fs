Module Tjr_minifs__Fuse_nfs

The most complicated example: an NFS client mounted via FUSE

This combines fuse with nfs client functionality (server does not need fuse of course).

NOTE nfs client ops could be in unix.m or lwt.m, although probably ocamlfuse does not work properly with lwt

Steps:

  • construct client
  • wrap with fuse

Essentially we compose the mk... functions, given some background types.
